Authentication

// Include your API key in the Authorization header const response = await fetch('https://api.myxenpay.finance/v1/merchants', { headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } });

API Endpoints

POST/merchants - Create a new merchant
GET/merchants/{id} - Get merchant details
POST/payments - Process a payment
GET/payments/{id} - Get payment status
POST/qr/generate - Generate QR code
GET/analytics - Get merchant analytics

Quick Start Example

// JavaScript/Node.js Example const MyxenPay = require('@myxenpay/sdk'); const client = new MyxenPay({ apiKey: 'your_api_key_here', environment: 'sandbox' // or 'production' }); // Create a payment const payment = await client.payments.create({ amount: 25.50, currency: 'USDC', merchantId: 'merchant_123', description: 'Coffee purchase' }); console.log('Payment created:', payment.id); // Generate QR code const qrCode = await client.qr.generate({ merchantId: 'merchant_123', amount: 25.50, currency: 'USDC' }); console.log('QR Code URL:', qrCode.url);
